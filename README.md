- 획득한 장비 아이템을 인벤토리에서 정리하면 어떨까하는 아이디어에서 시작
- 쉽고 간단한 로그라이크 게임
- 친숙하지만 약했던 몬스터인 슬라임을 주인공으로 설정
- 다양한 몬스터와 장비들로 게임 다양화
